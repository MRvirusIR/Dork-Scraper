<h1 align="center">Dork Scraper</h1>
<p align="center">Scrape website URLs using Google Dorks.</p>

# Dork_Scraper.py

## Requirements

```
pip3 install -r requirements.txt
```

## Usage

```
$ python3 Dork_scraper.py
> Enter Dork: inurl:index.php?id=
```
